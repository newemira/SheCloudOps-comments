# SheCloudOps Comments

This repository is used to store and manage comments for the SheCloudOps blog via [Giscus](https://giscus.app).

## How it works

- Each blog post has its own GitHub Issue for discussion.
- Comments are embedded in the Astro website using the Giscus script.
- Users need a GitHub account to post comments.

## Repository Structure

- `README.md`: this file
- `.gitignore`: ignores temporary files
- All comment management is handled via Giscus; no additional files are required.
